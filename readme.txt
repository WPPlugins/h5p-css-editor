== Description ==

= Alert =

Currently H5P CSS Editor only works with WordPress single site.  I am currently working on a solution but have not found one yet.


This plugin allow you to change the style of your H5P editor to match the style of your site.

This plugin requires the H5P plugin to work.

== Installation ==

From the WordPress plugin menu click on Add New and search for H5P CSS Editor 

Instead of searching for a plugin you can directly upload the plugin zip file by clicking on Upload:

Use the browse button to select the plugin zip file that was downloaded, then click on Install Now. The plugin will be uploaded to your blog and installed. It can then be activated.
